# Service User Preferences

Ce microservice gère les préférences des utilisateurs pour la personnalisation du contenu.

## Fonctionnalités

- Stockage et récupération des préférences utilisateur
- Gestion des catégories d'intérêt
- Mise à jour des préférences en temps réel
- Historique des modifications

## API Endpoints

- `GET /preferences/:userId` - Récupérer les préférences d'un utilisateur
- `POST /preferences/:userId` - Créer ou mettre à jour les préférences
- `PUT /preferences/:userId` - Modifier les préférences
- `DELETE /preferences/:userId` - Supprimer les préférences

## Port

Ce service fonctionne sur le port 3001
